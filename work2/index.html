About

A better hire is a web application that will track and manage interviews by tracking each individual candidate in a web application system manageable on your phone, computer, laptop or tablet where the candidates are ranked and tracked.

This web application was created by two students @ Codecore.ca a Vancouver based web programming bootcamp cocusing primarily on the Ruby on Rails language, but also touching on User Experience/User Interface, HTMl5, CSS, Javascript, Jquery, and SQL. This bootcamp will be concluding mid March 2014 and both designers/programmers will be available for hire after the completion of the course.

A better way to manage your interviews...

With an accurate & simple to use tracking system based in the Ruby on Rails programming language.
